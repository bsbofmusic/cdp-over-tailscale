# Tray Icons

Add these files to this directory:

- `connected.png` - 16x16, 32x32, 48x48, 256x256 PNG icon for connected state (green checkmark)
- `disconnected.png` - 16x16, 32x32, 48x48, 256x256 PNG icon for disconnected state (red X)
- `connected.ico` - Windows ICO file for connected state
- `disconnected.ico` - Windows ICO file for disconnected state

Also add:
- `../icon.png` - 512x512 app icon
